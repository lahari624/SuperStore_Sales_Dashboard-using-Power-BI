                                 Power BI dashboard project analyzing sales performance using visuals and DAX


PROJECT OVERVIEW :

This Sales Dashboard provides a comprehensive view of sales performance, helping stakeholders track revenue trends, regional sales distribution, and product performance.
It was built using Power BI Desktop, leveraging data visualization and DAX measures to uncover key business insights.


OBJECTIVES :

- Analyze total sales, profit, and quantity trends over time.
- Compare sales performance across regions, categories, and subcategories.
- Identify top-performing products and low-performing ones.
- Provide actionable insights through interactive visuals and filters.
  

TOOLS & TECHNOLOGIES USED
- Power BI Desktop
- Microsoft Excel / CSV (for dataset)
- Data Modeling and DAX
- Power Query Editor


KEY FEATURES :
- Dynamic dashboards with slicers for Region, Category, and Time Period.
- Visualizations include:
  
     • Bar and column charts for sales by region and category
  
     • Line charts for monthly sales trend

     • Donut chart for product category contribution
  
     • KPI cards showing total sales, profit, and quantity
  
     • Conditional formatting for high/low profit visualization
  
     • Interactive filters for easy data exploration.


DATASET DETAILS : 

Source: Kaggle 

Fields Used: Order ID, Order Date, Region, Category, Subcategory, Sales, Quantity, Profit

Rows: ~5877 records

INSIGHTS GAINED :

The West Region generated the highest revenue.

Technology Category contributed the most profit.

Sales showed seasonal patterns, peaking during Q4.

Identified products with high sales but low profit margins.

CONCLUSION :

This dashboard effectively showcases how Power BI can transform raw data into insightful business intelligence visuals.
It demonstrates end-to-end data cleaning, modeling, and visualization skills crucial for a Data Analyst role.
